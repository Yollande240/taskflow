# TaskFlow

## Description
TaskFlow est une application web de gestion de tâches permettant d'ajouter, afficher et supprimer des tâches.

## Technologies utilisées
- React
- Vite
- Node.js
- Express
- MongoDB

## Fonctionnalités
- Ajouter une tâche
- Afficher les tâches
- Supprimer une tâche

## Architecture
Frontend React → API Express → MongoDB
